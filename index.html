Engineering Projects & Technical Case Studies

This site presents selected BEng and MSc engineering projects focused on numerical modelling, CFD, and transient simulation problems. The work emphasises modelling decisions, solver configuration, and practical trade-offs, with the aim of translating academic analysis into industry-relevant engineering insight.

Technical Capabilities

  • Numerical Modelling & Simulation
    Compressible and incompressible CFD, multiphase flow modelling, transient simulations, mesh refinement strategies, solver configuration, and numerical stability considerations.
  • Programming & Scientific Computing
    Python (data analysis, scripting, machine learning workflows), MATLAB (numerical methods, signal processing), MATLAB/Simulink (dynamic systems, control, and aeroelastic modelling), Linux/Windows shell, HPC.
  • CFD & Analysis Tools
    Ansys Fluent/ICEM, ECOGEN (compressible multiphase CFD), SU2 (and other open source CFD), ParaView (post-processing and visualisation), structured and adaptive meshing, XML-based solver configuration.
  • Aerospace & Mechanical Engineering Applications
    Aerofoil analysis and preliminary aero design, aeroelastic and flutter analysis concepts, structural and fluid-structure interaction fundamentals.
  • Machine Learning & Data-Driven Methods
    Supervised learning fundamentals, neural networks for prediction tasks, feature engineering, and model evaluation in engineering contexts.
  • Computation & Tooling
    High-performance computing (HPC) workflows, batch job submission, resource and memory considerations, version control using Git/GitHub.

These capabilities are demonstrated across projects in CFD, aero design, aeroelastic modelling, and data-driven analysis, documented in the Projects section.
